# Ponytail

- YAGNI — don't add code until it's needed.
- Prefer Python/Node stdlib over pulling in a new dependency.
- Minimal code: fewer lines, fewer files, fewer concepts.
- No unnecessary abstractions: no base classes, factories, or interfaces unless there are two concrete implementations.
- Flat is better than nested. Simple is better than complex.
